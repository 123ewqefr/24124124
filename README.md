# 💳 Credit Agency Mod — Minecraft 1.16.5 Forge

Полноценная система кредитования для Minecraft 1.16.5 на базе Forge.

## ✨ Возможности

- 🏦 **НПС Кредитного агентства** — появляется по команде `/credit`, красивое оранжевое имя
- 📦 **Магазин кредитования** — все руды, оружие, броня всех материалов (70+ предметов)
- 📜 **Кредитный договор** — зачарованная книга с полными условиями кредита
- 💎 **Кредитный сундук** — ставьте на землю, кладите алмазы — долг погашается автоматически
- 📈 **Начисление процентов** — 3–10% каждую реальную минуту (сложные проценты)
- ⭐ **Кредитная история** — рейтинг от 0 до 400+, влияет на условия
- ☠️ **Коллекторы** — появляются через 5 игровых дней без платежа, неуязвимы для игрока
- 📊 **Команда `/credit history`** — история, текущий долг, рейтинг

---

## 🚀 Как собрать (GitHub Actions)

1. **Создайте репозиторий на GitHub** и загрузите все файлы из папки `minecraft-credit-mod/`
2. GitHub Actions **автоматически соберёт** мод при каждом push в `main`/`master`
3. **Скачайте JAR** из вкладки `Actions` → последний workflow → `creditmod-jar`
4. Поместите JAR в папку `mods/` вашего Minecraft Forge 1.16.5

### Ручная сборка (локально)

```bash
# Нужны: Java 8, Gradle 7.3
cd minecraft-credit-mod
gradle setupDecompWorkspace  # только первый раз (~10 минут)
gradle build
# JAR будет в build/libs/
```

## 📋 Как пользоваться кредитом

1. Введите `/credit` — рядом появится НПС с именем **Кредитное агентство**
2. Нажмите **ПКМ** по НПС — откроется меню кредитования
3. **ЛКМ** по предмету — выбрать/увеличить количество
4. **ПКМ** по предмету — подтвердить выбор и оформить
5. В инвентаре появится: товар + 📜 Кредитный договор + 💳 Кредитный сундук + 🚫 Отказ
6. Прочитайте договор, затем **кликните им по НПС** для подписания
7. Поставьте **Кредитный сундук** и кладите туда алмазы для погашения
8. Используйте предмет **«Отказ»** (барьер) на НПС, чтобы отменить кредит

## ⭐ Кредитная история

| Рейтинг | Статус | Ставка | Срок |
|---------|--------|--------|------|
| ≤ 50 | 🟢 Очень хороший | 3%/мин | 40 дней |
| 51–100 | 🟡 Хороший | 6%/мин | 20 дней |
| 101–200 | 🟠 Неплохой | 8%/мин | 10 дней |
| 201–300 | 🔴 Плохой | 10%/мин | 5 дней |
| 300+ | ⛔ Очень плохой | 10%/мин | 5 дней |
| > 400 | 🚫 Заблокирован | — | — |

### Изменение рейтинга
- Закрыть кредит в день взятия: **−30**
- Каждый хороший платёж: **−5**
- Платежи каждые 2 дня: **−2**
- Платежи каждые 3-4 дня: **−1**
- Просрочка 5+ дней: **+20**

---

## ☠️ Коллекторы

- Появляются через **5 игровых дней** без платежа
- **Неуязвимы** для атак игрока
- Если в инвентаре найдут алмазы — заберут их как платёж
- Через **10 игровых дней** без платежа — **изымается весь инвентарь**
